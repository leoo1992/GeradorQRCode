# Como Iniciar : 


## Passo 1: 

* Clonar esse repositório com:

```bash
git clone https://github.com/leoo1992/GeradorQRCode.git
```

## Passo 2: Iniciando o Aplicativo

```bash
# usando npm
npm install
npm start


# Ou usando Yarn
yarn add
yarn start
```

Agora você deverá ver seu o aplicativo em execução local no seu navegador.
